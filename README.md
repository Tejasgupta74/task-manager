# Team Task Manager (Full Stack)

## Features
- User Authentication (JWT)
- Role Based Access (Admin/Member)
- Project Management
- Task Assignment & Status Tracking
- Dashboard

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

## Backend Setup
```bash
cd backend
npm install
npm run dev
```

## Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Deployment
Deploy backend and frontend on Railway.
